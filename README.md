# Alke Wallet

En éste Proyecto del Módulo 2 | ABP
Estaré trabajando con los siguientes requerimientos:

## Objetivo
Desarrollar una billetera digital que permita:
- Registro e inicio de sesión
- Administración de fondos (depósitos y retiros)
- Envío y recepción de dinero
- Historial de transacciones

## Tecnologías
- HTML
- CSS
- JavaScript
- Bootstrap
- jQuery

## Estructura
- index.html
- login.html
- menu.html
- deposit.html
- sendmoney.html
- transactions.html

## Decisiones técnicas
- Bootstrap para responsividad y UI.
- jQuery para manipulación del DOM y validaciones.
- Rama principal: master (o main según corresponda).
